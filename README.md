# rust-ml

Machine learning library for the Rust programming language.

## Documentation

See [here](http://xasmx.github.io/rust-ml/doc/ml/index.html).

## Features

Features:
* classification
  * logistic regression
* clustering
  * k-means
* regression
  * linear regression

## Building

~~~
make
~~~

## Examples

To run examples: (You will need to have gnuplot installed to run them).

~~~
./out/kmeans
./out/linalg
~~~
